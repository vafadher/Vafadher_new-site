## Design Decisions

- **Typography**: Cormorant Garamond (display/editorial) + DM Sans (body) + JetBrains Mono (micro labels)
- **Color**: Deep graphite canvas, midnight slate surfaces, warm off-white text, steel-blue atmospheric glow, muted gold micro accents
- **Hero**: Asymmetric 42/58 split — editorial text left, operator console composition right
- **5 distinct composition modes**: Full-bleed editorial (Breakdown), Blueprint grid (Systems), Signal cards (Momentum), Step track (Working Rhythm), Memo/journal (Field Notes)
- **Motion**: Scroll-triggered reveals, subtle parallax, console bar animations, floating card depth
